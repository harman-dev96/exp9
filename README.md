## 2. Title

**Login Form with React State Management**

---

## 3. Approach

This experiment implements a responsive login form using React and modern state management techniques.

* A user interface is designed using Material UI components for a clean and responsive layout.
* Form handling and validation are implemented using **React Hook Form**.
* Email validation is performed using regular expressions to ensure correct format.
* Password validation enforces minimum length requirements.
* Error messages are displayed dynamically below input fields.
* A loading spinner is shown during form submission to simulate authentication processing.
* On successful validation, a success message is displayed to the user.
* The application uses React hooks (`useState`, `useForm`) to manage form state and UI feedback.
* The project is built and deployed using Netlify for easy accessibility.
